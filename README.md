# Prompt Builder — Mix Sora + Codex (V1)

Mini app web (HTML/CSS/JS) **offline-first** pour créer des prompts structurés.
- Templates (Codex + Sora)
- Variables (ex: {LIEU}, {STYLE}, {DUREE})
- Aperçu live / édition directe
- Historique + favoris
- Exports TXT / JSON

## Lancer
Ouvre `index.html` dans ton navigateur (iPhone/PC).

## Données
Stockées dans `localStorage` (sur ton appareil). Rien n'est envoyé.

## Fichiers
- `index.html`
- `styles.css`
- `app.js`
